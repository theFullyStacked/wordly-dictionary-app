# Worldly Dictionary App
## Description
This is a dictionary web app that allows users to search for words contained in an external API.
## Features
- A form to input the word
- Data fetching from the API
- Searching for the word
- Displaying the word's meanings, synonyms and antonyms, if any
- Error handling for missing input or invalid words
## How to run
1. Clone the repository
2. Open the project folder
3. Open the index.html file in your browser
## Technologies used
- HTML
- Javascript
- CSS
- Fetch API (Free Dictionary API)
## Future implementations
- Expand available vocabulary
- Add a loading spinner
- Improve UI styling
